日商簿記3級 模擬試験 Web App

这是一个基于 Web 的日商簿记3级模拟考试应用，旨在帮助考生练习仕訳（会计分录）题目。

🔗 在线访问: https://Joe-Xuu.github.io/boki/
✨ 功能特点

无需安装: 纯静态网页，打开即用。

随机抽题: 每次考试从题库中随机抽取 15道 题目，模拟真实考试节奏。

动态科目: 下拉菜单会根据当前题目动态显示相关的勘定科目，减少干扰。

借贷平衡: 模拟真实的借方/贷方输入界面，支持增加多行分录（复合仕訳）。

即时反馈: 提交后立即判分，并提供详细的题目解析。

🛠️ 技术栈

HTML5 / JavaScript (ES6+): 原生开发，无框架依赖，极速加载。

Tailwind CSS: 用于构建现代化的响应式用户界面。

Phosphor Icons: 简洁美观的图标库。

📂 文件结构

index.html: 应用的主程序逻辑和界面。

questions.js: 包含所有考试题目数据的题库文件。

📝 题目来源

题目改编自日商簿记3级的过往练习题与样题。
